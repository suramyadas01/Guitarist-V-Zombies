Guitarist V Zombies (ver. 1)

This is a fun game where the objective is to keep the hordes of zombies at bay while playing sick tunes. Use your guitar or any other instrument to create exact notes to control your character and defeat the incoming hordes of zombies.

The code and techical details about this project can be found on the Github link above.

Controls:

Guitar Controls Keyboard Controls
D5 - Up W - Up
E5 - Down S - Down
G5 - Shoot E - Shoot

Note: Guitar Controls signify which notes are to be played for the given actions. Feel free to have fun and add color and create your own compositions!!
